# Assets Folder

Place the following image files in this directory:

- `icon.png` - App icon (1024x1024 px)
- `splash.png` - Splash screen (1284x2778 px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)

You can use placeholder images from:
- https://placeholder.com
- https://via.placeholder.com
- Or create simple colored squares with your preferred design tool

For a quick start, you can use solid color squares:
- Icon: Blue square with white "SM" text
- Splash: White background with centered blue logo
- Adaptive Icon: Same as icon
- Favicon: Smaller version of icon
